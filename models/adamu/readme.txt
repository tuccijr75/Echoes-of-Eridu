Place your files for Adamu in this folder. The viewer will auto-detect by these names:
  - adamu.fbx  (or model.fbx)
  - albedo/basecolor/base_color.(png|jpg)
  - normal.(png|jpg)
  - roughness.(png|jpg)   [optional]
  - metallic.(png|jpg)    [optional, default is 0]
  - ao/occlusion.(png|jpg)[optional]
